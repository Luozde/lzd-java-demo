# docker-compose 基础

## 常用命令

```bash
# 创建编排docker-compose服务
docker-compose up

# 创建编排docker-compose服务, 后台启动
docker-compose up -d

# 停止docker-compose服务 不删除容器
docker-compose stop

# 停止docker-compose服务 删除容器
docker-compose down

# 启动docker-compose服务
docker-compose start

# 重启docker-compose服务
docker-compose restart

```
